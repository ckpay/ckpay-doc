- MD5签名规则
```
第一步：把字典按Key的字母顺序排序；
第二步：把所有参数名和参数值串在一起；
第三步：在第二步生成的参数串后加入KEY;
第四步：讲第三步得到的参数串进行MD5加密；
第五步：将MD5串转大写。
```
